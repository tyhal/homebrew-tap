# tyhal's homebrew tap

I even use it on [linux](https://www.ypsidanger.com/homebrew-is-great-on-linux/)

## Install

### Mine

* [**crie**](https://github.com/tyhal/klar) - Universal meta-linter using containerized execution

```shell
brew install tyhal/tap/crie
```

* [**klar**](https://github.com/tyhal/klar) - Structured json → clear output

```shell
brew install tyhal/tap/klar
```

### Others

- **[gotraceui](https://gotraceui.dev/)** – An efficient frontend for Go execution traces ( Dominik Honnef )

```
brew install tyhal/tap/gotraceui
```
